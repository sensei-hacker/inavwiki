## Upgrading from 2.0.0 / 1.9.X



New targets:

* Omnibus F4 Nano v6
* Matek F722SE
* DYS F4Pro v2
* Kakute F7
* Matek F411 RSSI
* Matek F411 with second software serial
* SP Racing F7 Dual
* Kiss FC
